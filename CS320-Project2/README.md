# Project-one-Project-Two-CS320

How can I ensure that my code, program, or software is functional and secure? :

This assignment had 3 service projects each of which required functional testing. Using unit tests to ensure that each aspect of the program functions correctly and can handle invalid inputs.

How do I interpret user needs and incorporate them into a program? :

In this assignment my “client” made a list for each of the service’s and what attributes were needed for each one. One example being that each service had a ID attribute that needed to be less than 11 characters and unique from each of the other ids in that service.

How do I approach designing software? :

When starting a new assignment, I will carefully read over each of the requirements, and then make a list of the primary goals for said assignment. Then, I look over these requirements, and I make a decision on what needs to be done for the site program to meet functionality requirements. Making sure to check the requirements occasionally as I complete the program, ensuring that I stay on track, and that work is sufficient.

## How to run tests

Run `mvn test` from the project root directory to execute the unit tests.
